# freecodecamp-forms-and-tables-html-practice
A collection of Forms and Tables HTML practice projects built while learning the FreeCodeCamp Responsive Web Design certification
## Workshop Hotel Feedback Form
- In this workshop, you will practice working with HTML forms by building a Hotel Feedback Form.
## Worshop Final Exams Table
- In this workshop, you will practice working with HTML tables by building a final exam table for a group of students.
